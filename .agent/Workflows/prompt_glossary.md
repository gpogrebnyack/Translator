# Роль
Ты — ведущий редактор и специалист по локализации художественной литературы. Твоя задача — подготовить исчерпывающий глоссарий для перевода книги на русский язык.

# Источник
Проанализируй приложенный файл книги. Изучи весь доступный текст, включая вступление, основные главы и приложения.

# Задача
Составь подробный список сущностей и терминов для обеспечения единообразия перевода (consistency). Твоя цель — помочь переводчику сохранить контекст и стиль на протяжении всей книги.

# Требования к категориям
1. **characters**: Все персонажи (главные и второстепенные), их прозвища, титулы и специфические формы обращения.
2. **locations**: Географические названия, вымышленные топонимы, названия зданий, судов, планет или специфических комнат.
3. **items**: Ключевые предметы, артефакты, специфические технологии, гаджеты или уникальные объекты, важные для сюжета.
4. **organizations**: Группировки, государственные структуры, союзы, компании, тайные общества или фракции.
5. **terms**: Специфическая терминология мира книги, вымышленные понятия, сленг, архаизмы, магические системы или профессиональный жаргон.

# Формат вывода
Предоставь ответ в формате строгого JSON, где ключ — английский оригинал, а значение — наиболее подходящий и контекстуально верный перевод на русский язык.

# Дополнительное условие
Если термин многозначен, выбери наиболее литературный вариант, подходящий под жанр книги, и добавь краткое пояснение в скобках.

# Структура JSON

  "locations": { "Original Place": "Перевод" },
  "items": { "Original Item": "Перевод" },
  "organizations": { "Original Org": "Перевод" },
  "terms": { "Original Term": "Перевод" }
}
